# 動くお絵かきAR — Hiroマーカー版

Hiroマーカー上で魚・クラゲ・自分でアップロードした絵を動かすWeb ARアプリです。静的ファイルだけで動くので、GitHub Pagesにそのまま置けます。

## 追加したこと

- Hiroマーカー印刷サイズを選択可能にしました（5cm / 7cm / 9cm / 12cm）
- AR内の絵の表示高さをスライダーで変更可能にしました
- デフォルト絵に「魚」と「クラゲ」を追加しました
- 自分で描いた画像への差し替えも残しています

## 使い方

1. `index.html` と `hiro-marker-print.html` をGitHubリポジトリ直下に置く
2. GitHubの `Settings` → `Pages` で `Deploy from a branch` を選び、`main` / `/root` を公開元にする
3. 公開URLをスマホで開く
4. `hiro-marker-print.html` でHiroマーカーのサイズを選んで印刷
5. アプリで魚・クラゲを選ぶ、または自分の絵をアップロードする
6. 高さスライダーで絵の浮き上がる高さを調整する

## 認識のコツ

小さいマーカーは便利ですが、認識距離が短くなります。うまく認識しない場合は、7cm〜9cm以上にする、明るい場所で使う、マーカー全体を画面に入れる、紙を折らない、を試してください。
