# 動くお絵かきAR - Hiro全画面録画版

Hiroマーカーを使って、魚・クラゲ・自分の絵をAR表示する静的Webアプリです。GitHub Pagesにそのまま配置できます。

## 今回の修正

- AR.jsのカメラ映像を画面いっぱいに表示します。
- カメラ映像はAR.jsの1系統だけにしました。
- 録画は、画面表示と同じ構成で「カメラ映像 + 絵」だけを合成します。
- 録画ボタンは普通のカメラアプリのように下部中央へ移動しました。
- 高さ・横位置スライダーは上部に配置しました。
- 大きさ・回転スライダーは右側に配置しました。
- menu、スライダー、トースト、録画ボタンは録画には入りません。

## 使い方

1. `index.html` をGitHub Pagesで公開します。
2. `hiro-marker-print.html` を開いてHiroマーカーを印刷します。
3. スマホで `index.html` を開き、カメラを許可します。
4. Hiroマーカーを映します。
5. 上と右のスライダーで、高さ・横位置・大きさ・回転を調整します。
6. 下部中央の赤い録画ボタンで録画を開始/停止します。
7. 停止後に表示される「動画を保存」から保存します。

## 注意

録画は `MediaRecorder` と `canvas.captureStream()` を使います。Android Chrome / Edgeでの利用がおすすめです。iOS Safariでは録画保存が制限される場合があります。
