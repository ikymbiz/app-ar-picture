# 動くお絵かきAR — Hiroマーカー版

Hiroマーカー上で、魚・クラゲ・自分でアップロードした絵を動かすWeb ARアプリです。静的ファイルだけで動くので、GitHub Pagesにそのまま置けます。

## 今回の修正

- 右端の黒帯対策として、表示用のカメラ映像を全画面 `cover` 表示に固定しました。
- AR.js内部のカメラvideoはトラッキング用に残し、画面表示は同じ映像を全画面表示する専用videoへ切り替えました。
- 位置パッドは非表示に戻しました。
- 絵そのものをドラッグして、Hiro基準の横位置・高さを調整する方式に戻しました。
- 絵はHiroマーカーの子要素としてAR追従します。
- 録画ボタンは下部中央のままです。
- 録画にはUIを含めません。

## 操作

- 絵をドラッグ: Hiroからの横位置・高さを調整
- 左下パッド: X/Y回転を調整
- 右下パッド: 幅/高さを調整
- 下部中央ボタン: 録画開始・停止
- 右上menu: 絵の選択、自分の絵への差し替え、Hiro印刷

## 今回の修正

- 録画が短時間で勝手に停止する問題を修正しました。
- Android / Chrome では安定性のため WebM 録画を優先します。
