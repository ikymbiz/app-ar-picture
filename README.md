# 動くお絵かきAR — Hiro Canvas単一カメラ版

Hiroマーカー上に、魚・クラゲ・自分でアップロードした絵を表示するWeb ARアプリです。GitHub Pagesにそのまま置けます。

## 今回の修正

- カメラ映像が2つに割れて見える問題を修正しました。
- 画面表示のカメラは AR.js が作る video 1つだけにしました。
- 自前の `cameraPreview` は録画フォールバック用に非表示化しました。
- 絵は透明canvasにだけ描画します。
- 録画用canvasには「カメラ映像 + 絵」だけを合成します。
- menu、スライダー、トースト、録画ボタンは録画に入りません。

## 使い方

1. `index.html` をGitHub Pagesで公開、またはHTTPS環境で開きます。
2. `hiro-marker-print.html` を印刷します。
3. カメラでHiroマーカーを映します。
4. 画面下で高さ・横位置・大きさ・回転を調整します。
5. 左上の録画ボタンで録画できます。
