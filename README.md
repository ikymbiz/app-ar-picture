# 動くお絵かきAR — Hiro追従録画版

Hiroマーカー上に魚・クラゲ・自分の絵を表示し、AR画面を録画できる静的Webアプリです。GitHub Pagesにそのまま配置できます。

## 今回の修正

- シャッター音を削除しました。
- 絵を画面固定ではなく、Hiroマーカーの子要素として配置しました。
- Hiroが動くと、絵もAR.jsのマーカー姿勢に直接追従します。
- 見失ったHiroの最後の位置に絵を固定し続ける処理をやめました。
- 録画ボタンは下部中央です。
- 高さは右側、回転は撮影ボタン左側、大きさは撮影ボタン右側で調整します。
- 録画停止後は自動ダウンロードします。
- マイク音声を録音します。
- MP4保存を優先します。ブラウザがMP4のMediaRecorderに未対応の場合はWebMで保存します。

## 使い方

1. `index.html` をGitHub Pagesで公開します。
2. `hiro-marker-print.html` を開いてHiroマーカーを印刷します。
3. スマホで `index.html` を開き、カメラ権限とマイク権限を許可します。
4. Hiroマーカーを映すと絵が出ます。
5. 下部中央の録画ボタンで録画開始・停止します。
6. 停止後、動画が自動でダウンロードされます。

## 注意

- iPhone / Android / ブラウザにより、MediaRecorderのMP4対応状況が異なります。MP4非対応の場合はWebMで保存されます。
- 絵はHiroマーカーを認識している間だけAR位置に追従します。


## 今回の修正

- 撮影ボタン左側の回転スライダーの上に「横幅」と「絵高さ」のスライダーを追加しました。

- 絵はHiroマーカーの子要素のままなので、画面固定ではなくHiroに追従します。
