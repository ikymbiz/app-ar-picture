# 動くお絵かきAR — コントロールパッド版

Hiroマーカー上に魚・クラゲ・自分の絵を表示し、画面を見ながら直感的に調整できるWeb ARアプリです。静的ファイルだけで動くため、GitHub Pagesにそのまま配置できます。

## 今回の実装

- 回転を2Dコントロールパッド化
  - 左右: Y軸回転
  - 上下: X軸回転
- 幅・高さを2Dコントロールパッド化
  - 左右: 横幅
  - 上下: 絵の高さ
- 絵そのものをドラッグして位置調整
  - 左右: Hiroマーカー基準の横位置
  - 上下: Hiroマーカー基準の表示高さ
- 絵は画面固定ではなくHiroマーカーの子要素として配置
- 録画ボタンは下部中央
- シャッター音なし
- 録画停止後に自動ダウンロード
- マイク許可がある場合は音声も録音
- MP4対応ブラウザではMP4、非対応時はWebMで保存
- 録画にはUI、menu、パッド、トースト、録画ボタンを含めません

## 使い方

1. `index.html` を開く、またはGitHub Pagesで公開します。
2. `hiro-marker-print.html` を開いてHiroマーカーを印刷します。
3. カメラでHiroマーカーを映します。
4. 絵をドラッグして、横位置と高さを調整します。
5. 左下の回転パッドでX/Y回転を調整します。
6. 右下のサイズパッドで幅/高さを調整します。
7. 下部中央の録画ボタンで録画開始・停止します。

## 注意

- カメラとマイクを使うため、GitHub PagesなどのHTTPS環境で開いてください。
- ブラウザによってはMP4録画に対応していないため、その場合はWebMで保存されます。
