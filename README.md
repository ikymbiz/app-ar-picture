# 動くお絵かきAR — Hiroマーカー版

Hiroマーカー上で、魚・クラゲ・自分でアップロードした絵を動かすWeb ARアプリです。GitHub Pages にそのまま置けます。

## 今回の修正

- 右端に黒帯が出ないよう、AR.js の video / canvas を全画面 cover 表示に固定しました。
- 上部に「位置 横/高さ」パッドを追加しました。
- 位置パッドの左右で Hiro からの横位置、上下で Hiro からの高さを調整できます。
- 絵を直接ドラッグしても、Hiro基準の横位置・高さを調整できます。
- 絵はHiroマーカーの子要素としてAR追従します。
- 録画ボタンは下部中央です。
- 録画にはUIを含めません。

## ファイル

- `index.html` アプリ本体
- `hiro-marker-print.html` Hiro印刷ページ
