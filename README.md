# 動くお絵かきAR — Hiro Canvas録画版

Hiroマーカーを使い、魚・クラゲ・自分の絵をカメラ映像の上に表示するWeb ARアプリです。静的ファイルだけで動くので、GitHub Pagesにそのまま置けます。

## 今回の修正

- カメラ映像が黒く隠れる問題を修正しました。
- AR.js / A-Frame のWebGL canvasは透明化し、カメラ映像は専用のプレビューvideoで必ず前面表示されるようにしました。
- 表示用canvasは透明背景で、絵だけを描画します。
- 録画用canvasには「カメラ映像 + 絵」だけを合成します。
- UI、menu、スライダー、トースト、録画ボタンは録画に入りません。
- 絵の縦横比は `幅 = 高さ × 元画像比率` で固定しています。
- 高さスライダーは絵のY位置だけを変更します。

## 使い方

1. `index.html` をHTTPS環境で開きます。GitHub Pagesならそのまま使えます。
2. カメラ許可を出します。
3. `hiro-marker-print.html` を開いてHiroマーカーを印刷します。
4. Hiroマーカーを映します。
5. 画面下のスライダーで高さ・横位置・大きさ・回転を調整します。
6. 左上の `録画開始` → `停止` → `動画を保存` で録画できます。

## 注意

- 録画は `MediaRecorder` と `canvas.captureStream()` を使います。Android Chrome / Edge で動作しやすいです。
- iPhone Safariでは録画保存が制限される場合があります。
