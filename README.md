# 動くお絵かきAR — Hiro true AR 録画版

Hiroマーカーに追従する3D ARオブジェクトとして、魚・クラゲ・自分の絵を表示します。
GitHub Pages にそのまま置ける静的Webアプリです。

## 今回の変更

- 絵を画面固定ではなく、Hiroマーカーに追従するARオブジェクトに戻しました。
- AR.js のカメラ画面を全画面表示します。
- 高さ調整は右側に配置しました。
- 回転調整は撮影ボタンの左側に配置しました。
- 大きさ調整は撮影ボタンの右側に配置しました。
- 撮影ボタンの赤丸と白枠がズレないように修正しました。
- 録画停止後、自動でダウンロードします。
- 録画開始時にシャッター音を鳴らします。
- マイク音声を動画に入れます。録画開始時にマイク許可が求められることがあります。
- MP4保存を優先します。ブラウザがMP4録画に未対応の場合はWebMで保存します。
- 録画されるのは AR.js のカメラ映像 + ARオブジェクトだけです。UIは入りません。

## 使い方

1. `index.html` をHTTPS環境で開きます。GitHub Pagesで公開できます。
2. `hiro-marker-print.html` を開き、Hiroマーカーを印刷します。
3. カメラ権限を許可します。
4. Hiroマーカーを映すと、絵がARとして表示されます。
5. 下中央の赤い撮影ボタンを押すと録画開始、もう一度押すと停止します。
6. 停止後、自動で動画がダウンロードされます。

## 注意

- 音声を入れるにはマイク権限が必要です。
- MP4で保存できるかはブラウザのMediaRecorder対応に依存します。
- MP4非対応の場合でも、録画自体はWebMで保存されます。
