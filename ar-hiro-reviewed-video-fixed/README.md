# 動くお絵かきAR — Hiroマーカー版（再レビュー修正版）

Hiroマーカー上で、魚・クラゲ・自分でアップロードした絵を動かすWeb ARアプリです。静的ファイルだけで動くので、GitHub Pagesにそのまま置けます。

## 再レビューで直した点

- 絵の表示を3D平面ではなく、画面上の2Dオーバーレイにしています。
- 絵の幅と高さを毎フレーム同じ計算式で明示するようにし、`height + width:auto` による端末差を避けました。
- 高さを変えても、カメラを遠ざけても、絵の縦横比は変わりません。
- Hiroマーカーから取得するのは画面上の位置だけです。マーカーの距離・傾き・検出スケールは絵の形に使いません。
- Hiro検出の細かい揺れを少し平滑化しました。
- Hiro通知は短時間・低い透明度・連発防止つきにしました。
- 録画canvasは、カメラ映像と絵だけを合成します。menu、スライダー、トースト、録画表示は入りません。
- 録画時も画面表示と同じ位置・大きさ・回転計算を使うようにしました。

## 使い方

1. `index.html` を開くか、GitHub Pagesに配置します。
2. `hiro-marker-print.html` を開いて、100% / 実際のサイズで印刷します。
3. カメラでHiroマーカーを映します。
4. 画面下のスライダーで、高さ・横位置・大きさ・回転をAR画面を見ながら調整します。
5. 右上の `menu` から魚・クラゲ・自分の絵を選べます。
6. `menu` → `録画開始` で録画できます。停止後に `動画を保存` を押してください。

## 注意

- 録画は `MediaRecorder` と `canvas.captureStream()` を使います。ブラウザによっては未対応の場合があります。
- iPhoneで録画が動かない場合は、Safariのバージョンや端末制限の可能性があります。その場合でもAR表示自体は使えます。
