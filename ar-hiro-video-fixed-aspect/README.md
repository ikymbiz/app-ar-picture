# 動くお絵かきAR — Hiroマーカー版

Hiroマーカーの位置を使って、魚・クラゲ・自分でアップロードした絵を表示するWeb ARアプリです。静的ファイルだけで動くので、GitHub Pagesにそのまま置けます。

## 今回の修正

- Hiroマーカー状態のトースト通知を小さく、短時間だけにしました。
- 絵の表示を3D平面から2Dオーバーレイ方式に変更しました。
- 高さを変えても、カメラを近づけたり遠ざけたりしても、絵の縦横比（x:y）は変わりません。
- 「高さ」は画像の変形ではなく、マーカー位置からの画面上の上方向オフセットとして動きます。
- 「大きさ」は均等拡大・均等縮小だけです。
- 動画撮影に対応しました。
- 録画される動画には、実際のカメラ映像と絵だけが入ります。menu、スライダー、トースト、録画表示は入りません。

## 使い方

1. `index.html` を開くか、GitHub Pages に配置して公開します。
2. `hiro-marker-print.html` を開いて、100% / 実際のサイズで印刷します。
3. カメラで Hiro マーカーを映します。
4. 画面下のスライダーで、高さ・横位置・大きさ・回転をAR画面を見ながら調整します。
5. 右上の `menu` から魚・クラゲ・自分の絵を選べます。
6. `menu` → `録画開始` で動画撮影できます。停止後、`動画を保存` から保存します。

## 注意

- 録画はブラウザの `MediaRecorder` に依存します。ブラウザによって保存形式は WebM または MP4 になります。
- Hiroを見失った後も、絵は最後に認識した画面位置に残ります。もう一度Hiroを映すと位置が更新されます。
- この版では絵を画面上の固定比率レイヤーとして出すため、3D平面のような遠近によるひしゃげは起きません。
