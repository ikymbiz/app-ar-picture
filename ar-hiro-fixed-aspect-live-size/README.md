# 動くお絵かきAR — Hiroマーカー版

Hiroマーカー上で、魚・クラゲ・自分でアップロードした絵を動かすWeb ARアプリです。静的ファイルだけで動くので、GitHub Pagesにそのまま置けます。

## 今回の修正

- 絵そのものの縦横比（x:y）を常に固定しました。
- Hiroの位置ズレや再検出の有無に関係なく、絵の比率は変わりません。
- 「大きさ」だけを均等拡大・均等縮小できます。
- 「高さ」は位置だけを変更し、画像の形は変えません。
- 高さ・横位置・大きさ・回転の調整パネル背景は透明のままです。
- Hiroマーカーを見失ってもトースト通知は表示しません。

## 使い方

1. `index.html` を開くか、GitHub Pages に配置して公開します。
2. `hiro-marker-print.html` を開いて、100% / 実際のサイズで印刷します。
3. カメラで Hiro マーカーを映します。
4. 画面下のスライダーで、高さ・横位置・大きさ・回転をAR画面を見ながら調整します。
5. 右上の `menu` から魚・クラゲ・自分の絵を選べます。
